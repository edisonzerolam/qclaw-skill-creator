﻿# qclaw-skill-creator

Modular, self-contained AI agent skill for the [OpenClaw](https://github.com/edisonzerolam/openclaw-skills) framework.

## Description

Guide and toolkit for creating new OpenClaw skills. Implements Anthropic's 4 hard constraints and provides engineering templates for skill design.

## Installation

Copy the qclaw-skill-creator/ directory to your OpenClaw skills directory (typically ~/.openclaw-skills/qclaw-skill-creator/).

## Structure

`
qclaw-skill-creator/
├── SKILL.md          # Main skill definition (read first)
├── references/       # Reference documentation
├── scripts/          # Executable tools
├── _knowledge/       # Knowledge base
└── _meta.json        # Skill metadata
`
